Tema 1: Tablas en HTML

Las tablas en HTML se utilizan para organizar y presentar datos en filas y columnas. Aquí tienes una descripción de los conceptos relacionados con las tablas:

  • Tabla en HTML: Una tabla HTML se compone de filas y columnas, similar a una hoja de cálculo. Cada celda individual está en la intersección de una fila y columna específica. Las tablas se crean con las etiquetas <table>, <tr> (fila), <td> (celda de datos) y <th> (celda de encabezado).
  • Formulario en HTML: Los formularios permiten a los usuarios enviar datos al servidor web. Se crean con la etiqueta <form> y contienen elementos como <input>, <select> y <textarea>.
  • Etiqueta table: Define una tabla.
  • Etiqueta form: Crea un formulario.
  • Etiqueta input: Utilizada para campos de entrada de datos en formularios.
  • Etiqueta tr: Representa una fila en una tabla.
  • Etiqueta label: Etiqueta para describir un elemento de formulario.
  • Etiqueta select: Crea una lista desplegable en formularios.
  • Etiqueta td y th: <td> representa una celda de datos, mientras que <th> es una celda de encabezado.
  • Etiqueta textarea: Permite la entrada de texto multilínea en formularios.
  • Etiqueta fieldset, legend: Agrupa elementos de formulario y proporciona una leyenda.
  • Etiqueta meter: Representa una medida escalar en un rango conocido.
  • Etiqueta thead: Define el encabezado de una tabla.
  • Etiqueta progress: Muestra el progreso de una tarea.
  • Etiqueta tfoot: Define el pie de una tabla.
  • Etiqueta datalist: Proporciona una lista de opciones para un campo de entrada.
  • Etiqueta tbody: Contiene el cuerpo de una tabla.
  • Etiqueta summary: Proporciona una descripción de una tabla para accesibilidad.

Tema 2: Estructuración de una Página Web

Los componentes esenciales para estructurar una página web son:

  • Contenedor (wrapper): Suele envolver la mayor parte del contenido de la página y define características básicas como anchura, bordes y alineación.
  • Cabecera (header): Contiene elementos invariantes de la parte superior de la página, como logotipos, imágenes o banners.
  • Contenido (content): Agrupa el contenido principal del sitio, como noticias, artículos o productos.
  • Menú (menu): Puede estar en la cabecera o en una barra lateral y proporciona opciones de navegación.
  • Pie (footer): Contiene información adicional, enlaces de contacto o derechos de autor.
  • Lateral (sidebar): Ubicado en los márgenes de la página, puede contener widgets, anuncios o enlaces relacionados.
  • Etiqueta div: Aunque no es semántica, se utiliza para agrupar elementos y aplicar estilos mediante CSS.

Tema 3: Estructuración Semántica de una Página Web

La semántica HTML5 permite definir la estructura de la página mediante elementos como <article>, <section>, <nav>, <aside>, <header> y <footer>. Estos elementos describen claramente su significado y propósito, tanto para los humanos como para las máquinas.
